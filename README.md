# ✍️ Handwritten Digit Classifier using CNN (MNIST)

This project uses a Convolutional Neural Network to classify handwritten digits from the MNIST dataset, achieving over 98% accuracy.

## 🚀 Features
- CNN with Conv2D, MaxPooling, Dense layers
- Data augmentation for better generalization
- Accuracy > 98%
- Confusion matrix & visual metrics

## 🔧 Tech Stack
- Python
- TensorFlow / Keras
- Google Colab
- Matplotlib

## 🛠️ How to Run
1. Open `mnist_cnn.ipynb` in Google Colab.
2. Run all cells to train and test the model.

## 📊 Output
- Accuracy curve
- Confusion matrix
- Sample predictions

## 📁 Files
- `mnist_cnn.ipynb` – Main notebook
- `README.md` – Project overview
